Console Security & Setup Instructions
Gmail Integration Setup
  1. Log in to your Sender Gmail Account.
  2. Navigate to Google Account -> Security.
  3. Enable **2-Step Verification** (required for app passwords).
  4. Search for **App Passwords** or navigate under "Signing in to Google".
  5. Generate a new app password (select App: "Mail", Device: "Windows Computer" or "Other").
  6. Copy the 16-character code and paste it into the **Gmail App Password** input.
Monitored directories (FIM)

Ensure directories configured under FIM path lists have correct Windows DACLs (NTFS permissions). Restrict modification accesses to trusted system users only. Watchdog FIM handles recursive modifications.

Session management guidelines

Sessions expire after **2 hours** of idle states automatically. Always log out of the operator console when leaving workstation desks. Access events are audited in the User Management log tab.
